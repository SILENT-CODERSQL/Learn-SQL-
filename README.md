# Learn-SQL-
Learn About SQL 
Is repository me ek advanced SQL interview problem ka solution diya gaya hai jisme Odd aur Even numbers ko identify kiya gaya hai without using %, / aur & operators. Ye problem interviews me commonly poochi jaati hai kyunki ye sirf syntax nahi balki logical understanding of SQL ko test karti hai.

Solution T-SQL (SQL Server) me likha gaya hai aur CTE / virtual table ka use karta hai, isliye kisi permanent table ki requirement nahi hoti. Query lightweight hai, interview-ready hai, aur directly SQL Server me run ki ja sakti hai.

Ye repository un candidates ke liye helpful hai jo SQL interview preparation, logical SQL puzzles, aur company-specific questions (TCS, Wipro, Infosys, HCL) practice karna chahte hain. Aap is code ko modify karke apni understanding aur strong bana sakte ho.
